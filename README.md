This repository is aimed for those who are newbie to Asterisk's new SIP stack named PJSIP (res_pjsip).

Visual high view on PJSIP objects could help you quicker understand the stack.

Diagram was created based on information from:
- pjsip.conf sample file
- `config show help res_pjsip` CLI commands
- https://wiki.asterisk.org
